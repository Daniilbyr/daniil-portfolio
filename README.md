# Портфолио Даниил Кузовлев

Одностраничник с нуля: только HTML/CSS/JS, без сборщика.

## Локально

Откройте `index.html` в браузере или:

```bash
npx --yes serve .
```

## GitHub Pages

1. Создайте репозиторий, например `daniil-portfolio`.
2. Залейте содержимое этой папки в корень или в `/docs`.
3. Settings → Pages → ветка `main`, папка `/ (root)` или `/docs`.
4. Ссылка будет: `https://<user>.github.io/daniil-portfolio/`

В откликах и резюме используйте эту ссылку вместо чужих сайтов.
