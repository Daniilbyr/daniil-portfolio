# Демо-лендинг под рекламный трафик

Статическая вёрстка: **3 блока** (hero → преимущества → заявка), **mobile-first**, **липкая CTA** на мобилке, заготовка под **Яндекс.Метрику** и цели `reachGoal`.

## Быстрый просмотр

Откройте `index.html` в браузере или поднимите локально:

```bash
npx --yes serve .
```

## Публикация (GitHub Pages)

1. Создайте репозиторий, залейте содержимое папки.
2. Settings → Pages → Source: **Deploy from a branch** → branch `main`, folder `/ (root)`.
3. Ссылка будет вида: `https://ВАШ-НИК.github.io/ИМЯ-РЕПО/`

## Метрика

1. В `index.html` раскомментируйте блок Метрики и подставьте ID счётчика.
2. В `app.js` задайте `YM_ID` тем же числом.

## Состав

- `index.html` — разметка
- `styles.css` — стили
- `app.js` — скролл, цели, форма
