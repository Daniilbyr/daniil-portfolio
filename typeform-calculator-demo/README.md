# Демо квиза-калькулятора (портфолио)

**Зачем:** показать заказчику логику многошаговой формы и настраиваемые цены без доступа к их Typeform.

- **Цены и параметры** — в `config.js` (один файл).
- **Иконки** — простые SVG в коде, не заимствованы из чужих калькуляторов.
- **Публичная ссылка** (после push):  
  `https://daniilbyr.github.io/daniil-portfolio/typeform-calculator-demo/`

## Плюс к этому демо

Сделайте в Typeform **отдельную** короткую форму (квиз 5–7 шагов с логикой) и приложите **прямую ссылку Typeform** — так закрывается требование «опыт в Typeform» + есть техдемо.
